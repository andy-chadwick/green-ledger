# The Green Ledger — Design Brainstorm

## Design Philosophy

Three distinct approaches considered:

<response>
<text>
**Approach A: Investigative Dossier**
Design Movement: Cold War intelligence file meets modern data journalism (NYT/Guardian interactive)
Core Principles: Credibility through restraint; data as the hero; dark authority; typographic hierarchy
Color Philosophy: Near-black charcoal (#0f1117) background, off-white body text, acid green (#22c55e) as the only accent — the Green Party's own colour weaponised as critique
Layout Paradigm: Asymmetric editorial grid. Wide left column for policy narrative, narrow right column for data callouts and case study flags
Signature Elements: Red/green risk dials; "CLASSIFIED" stamp aesthetic on case studies; monospaced data figures
Interaction Philosophy: Deliberate, weighty interactions. Cards expand with a slow reveal. No frivolous animation.
Animation: Staggered fade-in on scroll; number counters animate on entry; chart bars draw left-to-right
Typography System: Playfair Display (headlines) + IBM Plex Mono (data/figures) + Inter (body)
</text>

**Approach B: Parliamentary Dispatch Box** (CHOSEN)
Design Movement: British parliamentary gravitas meets The Economist's visual language
Core Principles: Authoritative but accessible; structured argument; evidence-first; British institutional feel
Color Philosophy: Deep parliamentary green (#1a3a2a) as the dominant dark tone, warm cream (#f5f0e8) for content areas, sharp red (#dc2626) for risk indicators, gold (#d4a017) for data callouts. The palette evokes the House of Commons chamber itself.
Layout Paradigm: Full-bleed hero with a UK map anchor; below, a masonry-style policy grid that breaks the standard column layout. Each card occupies different visual weights depending on fiscal risk score.
Signature Elements: Risk "thermometer" gauges; "Case Study" stamps with country flags; split-panel "They Say / We Say" debate cards
Interaction Philosophy: Each policy card is a self-contained investigation. Clicking opens a full modal with tabbed sections. The calculator is a persistent bottom drawer.
Animation: Cards lift on hover with subtle shadow; modal slides up from bottom; chart values count up on tab activation; map regions pulse on hover
Typography System: Libre Baskerville (headlines — authoritative serif) + Source Sans 3 (body — readable) + JetBrains Mono (data figures)
</text>

**Approach C: Brutalist Broadsheet**
Design Movement: Swiss International Typographic Style meets digital brutalism
Core Principles: Confrontational honesty; no decoration; pure information architecture; typographic boldness
Color Philosophy: White ground, black type, single accent of bright green. Inspired by political pamphlets.
Layout Paradigm: Rigid column grid with deliberate rule breaks. Large bold numbers dominate sections.
Signature Elements: Oversized pull-quotes; stark horizontal rules; bold numbered policy list
Interaction Philosophy: Fast, no-nonsense. Click = instant reveal. No animation beyond necessity.
Animation: Minimal. Instant transitions. Bold text flashes on key data points.
Typography System: Space Grotesk (all weights) — single typeface, maximum contrast through weight variation

## Chosen Approach: B — Parliamentary Dispatch Box

Deep parliamentary green background, cream content areas, red risk indicators, gold data callouts. Libre Baskerville headlines, Source Sans 3 body, JetBrains Mono for all data figures. Asymmetric layout with UK map hero, masonry policy grid, tabbed modal deep-dives, and a persistent cost calculator drawer.
